# **КОММЕРЧЕСКОЕ ПРЕДЛОЖЕНИЕ**
*Разработка чат-ботов для Telegram и Max с интеграцией в Bitrix24*
## **1. Объём работ**

|**Наименование работ**|
| :- |
|Разработка чат-бота для Telegram (базовая логика, контент, полезные ссылки)|
|Разработка и адаптация чат-бота для платформы Max|
|Создание системы уникальных QR-кодов для n спикеров (deep linking)|
|Интеграция с Bitrix24 (настройка полей, передача данных, запись клиентов)|
## **2. Техническая реализация**
Многоканальность: Бот будет работать параллельно в двух мессенджерах (Telegram и Max). Вся информация и ссылки будут синхронизированы.

Система QR-кодов: Для каждого спикера генерируется уникальная ссылка, «зашитая» в QR. При переходе бот мгновенно определяет ID источника (кто именно из спикеров привёл клиента) и сохраняет его.

**Интеграция с Bitrix24:**

- Сбор контактных данных через интерфейс бота.
- Автоматическое создание лида/сделки в CRM через вебхуки.
- Передача информации об источнике (спикере) в соответствующее поле карточки Bitrix24.
